# Deep Research

Conduct thorough, systematic research on complex topics using multiple sources and perspectives.

## When to Use

- User needs comprehensive understanding of a complex topic
- Evaluating technologies, tools, or approaches
- Investigating best practices for a specific domain
- Answering questions that require deep analysis

## Research Process

### Phase 1: Scope Definition
- Define the research question precisely
- Identify sub-questions to answer
- Set boundaries (what's in scope, what's not)
- Determine desired output format

### Phase 2: Source Discovery
- Search for authoritative sources (documentation, papers, specs)
- Find expert opinions and case studies
- Look for contrasting viewpoints
- Identify the most current information

### Phase 3: Information Gathering
For each sub-question:
1. Find 3+ reliable sources
2. Extract key facts and data points
3. Note areas of consensus and disagreement
4. Record source citations

### Phase 4: Analysis
- Synthesize findings across sources
- Identify patterns and themes
- Evaluate strength of evidence
- Form conclusions based on evidence
- Note gaps in available information

### Phase 5: Report

```markdown
# Research Report: [Topic]

## Executive Summary
[2-3 paragraph overview of key findings]

## Background
[Context and why this matters]

## Key Findings

### [Finding 1]
[Evidence and analysis]

### [Finding 2]
[Evidence and analysis]

## Analysis
[Synthesis of findings, patterns, implications]

## Recommendations
1. [Actionable recommendation based on findings]
2. [...]

## Limitations
- [What this research doesn't cover]
- [Areas needing further investigation]

## Sources
- [Source 1]
- [Source 2]
```

## Quality Standards
- Cite sources for factual claims
- Present multiple viewpoints on controversial topics
- Distinguish between facts, opinions, and speculation
- Acknowledge uncertainty and limitations
- Prioritize recent information over older sources
- Verify critical claims from multiple sources
