# Bolder

Strengthen visual design to create more impactful, attention-grabbing interfaces.

## When to Use

- UI feels bland, generic, or forgettable
- Need stronger visual hierarchy
- User asks to "make it pop" or "make it bolder"
- Improving visual impact without changing layout

## Techniques

### Typography Impact
- Increase heading size contrast (2-3x larger than body)
- Use font weight contrast (bold headings, regular body)
- Add letter spacing to uppercase labels
- Use a distinctive display font for hero text
- Limit to 2 font families max

### Color Intensity
- Increase saturation of primary colors
- Use darker backgrounds with light text for drama
- Add gradient overlays for depth
- Use accent colors sparingly but vibrantly
- Ensure sufficient contrast ratios (WCAG AA minimum)

### Spacing & Scale
- Increase whitespace around key elements
- Use larger padding in cards and containers
- Scale up icons and illustrations
- Add generous margins between sections
- Let important elements breathe

### Visual Hierarchy
- Make CTAs unmistakably prominent
- Dim secondary content (lower opacity, smaller size)
- Use cards with shadows to elevate key content
- Add borders or dividers to create clear sections
- Use size, color, and position to guide the eye

### Personality
- Replace generic stock imagery with distinctive visuals
- Add subtle patterns or textures to backgrounds
- Use iconography consistently
- Consider dark mode for dramatic effect
- Add hover effects that reward exploration

## Application Order
1. Typography (biggest impact for least effort)
2. Color (second most impactful)
3. Spacing (improves readability and focus)
4. Visual accents (shadows, gradients, effects)
5. Imagery (most effort but most distinctive)
